# OpenTray Windows 文章术语

更新时间：2026-07-18

| 英文或用户表达                         | 文中解释                                                                     |
| -------------------------------------- | ---------------------------------------------------------------------------- |
| HWND                                   | Windows 窗口对象的句柄。它标识原生窗口，不等于页面组件或像素缓冲区。         |
| client area                            | 客户区，应用负责显示内容的区域。                                             |
| non-client area                        | 非客户区，标题栏、边框、系统菜单和原生 resize 区域。                         |
| WndProc                                | 接收并处理 Windows 窗口消息的函数。                                          |
| DWM                                    | Desktop Window Manager，收集窗口表面并合成最终桌面的系统组件。               |
| redirection surface                    | DWM 为窗口保留的离屏合成表面，可理解为窗口提交给桌面合成器的内容。           |
| Windowed WebView2                      | 以原生 child HWND 承载 WebView2 的默认 hosting 模式。                        |
| DirectComposition                      | Windows 的视觉树合成 API；使用 CompositionController 时还要自行处理输入。    |
| Mica / Acrylic                         | DWM system backdrop，不是 CSS 背景或 WebView2 自己的滤镜。                   |
| host paint                             | 顶层 HWND 对自己客户区的原生绘制责任。                                       |
| Black host paint                       | 用 `BLACK_BRUSH` 填满材质窗口客户区，作为 DWM 合成底，不表示最终画面是黑色。 |
| `WS_CLIPCHILDREN`                      | 父窗口绘制时避开子窗口区域；透明 WebView2 场景下会留下未被父窗口拥有的区域。 |
| `WM_NCCALCSIZE`                        | Windows 计算客户区和非客户区边界时发送的消息。                               |
| `DWMWA_VISIBLE_FRAME_BORDER_THICKNESS` | DWM 报告的可见窗口边框厚度，用于修正 frameless 顶部间距。                    |
| COM ordering                           | WebView2 与 AppWindow 等 COM API 的线程初始化顺序。                          |
| operational visibility                 | 原生窗口既未隐藏也未最小化，用户当前可以操作。                               |
| WebView2 profile                       | WebView2 的 user data folder，包含缓存、Cookie、权限和进程锁。               |
| WebView 只占 1/9，外部仍有残影         | 证明异常不受 WebView 覆盖范围约束，应检查顶层 HWND/DWM。                     |
| Win7 外圈                              | 非客户区旧 frame 残影，与客户区材质残影分开诊断。                            |
| retained session                       | 保留式窗口会话，隐藏窗口但继续保留 HWND、WebView 和页面状态。                |
| bootstrap show                         | 只负责第一次创建和装载的 `show()`，不能当作每次恢复动作。                    |
| single-flight drain                    | 共享单飞事件轮询，同一时刻只有一个 broker drain 请求。                       |
| switcher projection                    | 由 `showInSwitchers` 控制任务栏和 Alt+Tab 投影。                             |
| page-owned dismissal                   | 页面负责退出动画，设置 `autoHide: false`，结束后调用原生 `close()`。         |
